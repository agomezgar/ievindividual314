Diffraction of Light by a Single Slit

 

 
Condition for a maximum (approximation):
 
a = 0°    or    d sin a  =  (k + ½) l

b ... width of slit
a ... angle
k ... order of the maximum (1, 2, 3, ...)
l ... wavelength

 
Condition for a minimum:
 
d sin a  =  k l

b ... width of slit
a ... angle
k ... order of the minimum (1, 2, 3, ...)
l ... wavelength
